# Faintly

Distraction-free writing app. Everything you type fades away — keeps you moving forward.

Built with [Tauri 2](https://v2.tauri.app) + React + Tailwind.

## Use it

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```

The binary will be in `src-tauri/target/release/`. No server needed — it's a standalone desktop app.
